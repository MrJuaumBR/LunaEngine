# UI Layouts

Layouts help organize UI elements automatically.

Instead of manually positioning every element, use layout managers.

## Example

```python
    from lunaengine.ui.layout import VerticalLayout

    layout = VerticalLayout()
    layout.add(button1)
    layout.add(button2)
```

## Benefits

- easier organization
- automatic spacing
- responsive menus

Layouts are ideal for settings screens, inventories, and menus.
